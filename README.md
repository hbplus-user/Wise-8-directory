# Wise8 - The Future of Intelligent Living

Wise8 is a premium, high-performance web experience built for the modern era. Leveraging React, TypeScript, and Framer Motion, it delivers a seamless, interactive, and visually stunning journey.

## 🚀 Technologies

- **Core**: [React 19](https://react.dev/)
- **Logic**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: Vanilla CSS with [Glassmorphism](https://glassmorphism.com/) principles
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Build Tool**: [Vite 8](https://vitejs.dev/)

## ✨ Key Features

- **Dynamic Navigation**: Responsive and elegant top-tier navigation.
- **Immersive Hero**: High-impact visual landing with custom cursor interactions.
- **SwitchSection**: Interactive content switching for multi-dimensional storytelling.
- **Pathways**: Detailed exploration pages for various project facets.
- **Vision & Volumes**: Articulated project goals and technical depth.
- **Scroll Reveals**: Smooth, physics-based scroll animations.

## 🛠️ Getting Started

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## 📐 Project Structure

- `src/components`: Reusable UI components.
- `src/pages`: Main page layouts and routing.
- `src/assets`: Visual assets and icons.
- `src/index.css`: Core design system and global styles.
